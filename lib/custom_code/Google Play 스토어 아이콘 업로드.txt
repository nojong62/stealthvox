# 1️⃣ Google Play 스토어 아이콘 업로드 위치

경로:

```text id="hkrb0m"
Google Play Console
→ 게시 개요
→ 기본 스토어 등록정보
```

여기서 업로드:

| 항목                      | 규격               |
| ----------------------- | ---------------- |
| 앱 아이콘                   | 512x512 PNG      |
| 대표 그래픽(Feature Graphic) | 1024x500 PNG/JPG |
| 스크린샷                    | 폰 캡처             |
| 프로모션 영상                 | 선택               |

---

# 2️⃣ Play Store 아이콘 규격 ⭐

Google Play용은:

| 항목   | 값         |
| ---- | --------- |
| 크기   | 512 x 512 |
| 형식   | PNG       |
| 배경   | 투명 금지     |
| 최대용량 | 1024KB    |

중요:

```text id="hcvnzw"
투명 배경 안 됨
```

즉:

* 완전 검정 배경
* 노란 로고

처럼 꽉 찬 아이콘 추천.

---

# 3️⃣ Adaptive Icon 구조 (안드로이드 런처용) ⭐⭐⭐

Android 8+부터 사용.

구조:

| 레이어        | 설명        |
| ---------- | --------- |
| Foreground | 실제 로고     |
| Background | 배경색/배경이미지 |

예:

* 배경 = 검정
* 전경 = 노란 실드/Vox 아이콘

---

# 권장 크기

| 요소         | 권장              |
| ---------- | --------------- |
| Foreground | 432x432 이상      |
| Background | 432x432 이상      |
| 전체 캔버스     | 1024x1024 제작 추천 |

---

# 중요한 안전영역

Adaptive Icon은:

* 원형
* 사각형
* squircle

로 잘립니다.

그래서:

```text id="3wqjbi"
중앙 66%
```

안에 핵심 로고 넣는 게 중요.

---

# 4️⃣ Android 13 Monochrome Icon ⭐⭐⭐

Android 13 이상은:

* 테마 아이콘
* 단색 아이콘

지원합니다.

필수는 아니지만 추천.

---

# 규칙

| 항목     | 설명       |
| ------ | -------- |
| 단색     | 흰색 또는 검정 |
| 그림자 금지 | 단순 벡터    |
| 배경 없음  | 투명       |

예:

* 흰색 실드 로고
* 흰색 V

---



# 5️⃣ 스플래시 최적화 ⭐⭐⭐

StealthVox 스타일 추천:

| 요소    | 추천     |
| ----- | ------ |
| 배경    | 완전 블랙  |
| 로고    | 노랑/화이트 |
| 애니메이션 | 최소     |
| 표시시간  | 1~1.5초 |

# 추천 구성

## 배경

```text id="7x3y9u"
#000000
```

## 로고

* 중앙 정렬
* 화면의 22~28%

# 절대 피할 것

| 문제      | 이유       |
